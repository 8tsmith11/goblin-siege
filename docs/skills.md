# Player Skills

Player skills are global upgrades that apply to your entire run. Open the skill panel by clicking the **⚡ skill points** counter in the HUD.

- Earn **+1 Skill Point** every 3 waves, and occasionally from random events.
- **Tier 2** requires **2 Tier 1 skills** owned first.
- **Tier 3** requires **2 Tier 2 skills** owned first.

---

## Tier 1 — Cost: 1 ⚡ each

| Skill | Effect |
|---|---|
| **Sharp Shot** | +20% damage to all currently placed towers (applied immediately) |
| **Thick Walls** | +5 lives immediately |
| **Gold Rush** | +30% factory income per wave |
| **Quick Draw** | −15% cooldown on all currently placed towers (applied immediately) |
| **Spell Master** | −25% cost on all spells |

## Tier 2 — Cost: 2 ⚡ each (requires 2× Tier 1)

| Skill | Effect |
|---|---|
| **Bee Keeper** | +2 bees per Beehive support structure |
| **Doom Laser** | Factory lasers deal 2× damage |
| **Greed** | +3 gold per enemy kill |
| **Clown Master** | Clown support reverse duration lasts 2× as long |

## Tier 3 — Cost: 3 ⚡ each (requires 2× Tier 2)

| Skill | Effect |
|---|---|
| **Robot Overclock** | AI Agent casts spells twice as fast (every 150 ticks instead of 300) |
| **Mega Factory** | All factory income ×1.5 (stacks with Gold Rush) |

---

## Tower Skills

Each individual **placed tower** also has its own A/B/C skill tree, separate from player skills. Click a placed tower and press **Skill** to open it.

- **A and B are mutually exclusive** — you can only pick one.
- **C requires either A or B** to be purchased first.
- These skills cost **Skill Points ⚡** from the shared pool.

See `towers.md` for the full per-tower skill trees.
