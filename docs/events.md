# Random Events

During the **active phase**, a random event has a **40% chance** of triggering mid-wave (after wave 1). Events appear as a banner on screen for 3 seconds.

There are 10 possible events — 7 good and 3 bad.

| Event | Type | Effect |
|---|---|---|
| 💰 Gold Rush! | Good | +50 gold |
| ❤️ Reinforcements! | Good | +3 lives |
| ⚡ Power Surge! | Good | All towers +5 DMG this wave |
| 🏭 Overtime! | Good | Immediately grants bonus gold equal to one wave of factory income |
| 🐝 Bee Frenzy! | Good | All bees fire 2× faster for the rest of the wave |
| ⚡ Skill Point! | Good | +1 free skill point |
| 💀 Goblin Ambush! | Bad | 5 fast goblins spawn immediately at the path start |
| 🌑 Darkness! | Bad | All enemies on the map go stealth for 3 seconds (180 ticks) |
| 💸 Tax Collector! | Bad | −30 gold (cannot go below 0) |
| 🔥 Wildfire! | Bad | A random non-factory tower is disabled for 1 wave |

## Notes

- **Power Surge** is a one-time boost to current tower damage values — the +5 carries forward permanently for those towers.
- **Overtime** grants exactly the same gold you would earn at end-of-wave from your factories (including monkey and skill multipliers).
- **Bee Frenzy** cuts bee cooldowns in half. Bees that are destroyed before the wave ends still benefit for their remaining life.
- **Wildfire** picks a random combat tower (not factory). The affected tower resumes firing next wave.
- **Goblin Ambush** enemies use wave-scaled stats and spawn at the very start of the path.
