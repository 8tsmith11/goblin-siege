## v1.3 — Feed Log & Inventory Stacking

### Game Feed Log
- New sidebar feed panel shows a live log of everything that happens — wave starts, boss arrivals, random events, research completions, and Scribe's Journal entries as they unlock
- Feed is saved and restored with the game so nothing is lost on reload
- Scrollable history; auto-scrolls to the bottom unless you've scrolled up to read

### Inventory Stacking
- Augments and consumables now stack by type instead of taking a separate slot each
- Stack count shown as a badge on inventory cells and on the action button in the panel

### Crafted Item Delivery
- Crafted items delivered to a Stockpile in interface mode now route directly to your inventory instead of the resource pool
- Workbench stock panel shows a deposit (+1) button next to each resource so you can fund crafts without needing a monkey
- Stockpile tooltip correctly handles withdrawal of crafted items (augments → augment inventory, consumables → consumables inventory)

### Bug Fixes
- Consuming an augment or consumable from a stacked inventory entry no longer wipes the entire stack
- Monkey carry icon now displays correctly for crafted item types
- HUD resource counts update in-place to preserve any attached event listeners
