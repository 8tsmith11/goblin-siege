## v1.1 — Resources & Polish

### Resources
- Stone nodes (🪨) placed on the map each game — click to mine with a 20% drop chance
- Wood (🪵) added as an inventory resource, dropped by enemies
- Ground loot stacking system — items drop on tiles, up to 4 stacks of 64 per tile
- Resources displayed in the HUD next to gold

### Save System
- Auto-save at the end of each wave to browser storage
- Export save file (💾) and import save file (📂) buttons in the HUD
- Restart button (🔄) with optional export-before-wipe checkbox
- Obfuscated save format with integrity checksum

### Prep Phase
- 30-second build phase between every wave (not just the first)
- Mining and all support systems active during prep

### Bug Fixes
- Freeze spell no longer pauses towers, factory lasers, or other systems — only enemy movement is halted
- Path visual now always matches the actual enemy route (stale background cache fixed)
- Tower rendering no longer offset after scrolling
- Castle scaled down
