## v1.0 — Initial Release

### Towers & Combat
- 11 unique tower types, each with randomized stat upgrades
- Per-tower skill trees with A/B/C upgrade paths, unlocked with skill points
- Support structures: Clam (damage aura), Beehive, Clown (reversal), Monkey (factory boost), Robot (auto-cast)
- Factory system for passive gold income each wave

### Enemies
- Multiple enemy types with varied speeds, HP, and resistances
- Poison and healing enemy behaviors
- Boss waves every 5 rounds with unique dialogue and big rewards

### Progression
- Procedurally generated winding maze path — different every game
- Skill tree with 20+ permanent upgrades, earned every 3 waves
- Random mid-wave events that shake up the battlefield

### Spells
- Volcano — erupts a tile in sustained area damage
- Freeze — halts all enemies for several seconds
- More spells unlockable through the skill tree

### Audio
- Procedural music engine that evolves with the wave number
- Sound effects for combat, placement, and boss encounters
