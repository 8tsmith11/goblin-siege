## v1.8 — Skill Reworks & UI Polish

### Skill Reworks

**Lion**
- New skill D: **Lion's Pride** — +30% DMG per adjacent Lion (diagonals count). Stacks per neighbor.
- Replaces Relentless (+35% attack speed)

**Penguin** (full rework — no splash on base tower)
- A: **Cryo Shell** — shots explode on impact (splash 1.2)
- B: **Glacial Bite** — slow increased to 80% (mutually exclusive with A)
- C: **Lingering Chill** — slowed enemies permanently retain 25% slow after it expires; not stackable
- D: **Brittle Ice** — +80% DMG to enemies that are already slowed

**Clown**
- B **Pratfall** stun reduced from 25 → 10 ticks
- C **Jester's Privilege** (reworked) — swaps the globally frontmost and backmost enemy; both rush to each other's position with a dramatic confetti burst; clown still reverses its primary target as normal

### Balance

- Starting gold increased from 80 → 100
- Wave 5 (Proud Herald) HP reduced (~30% lower)
- Boss HP formula tuned down across all waves for a fairer early–mid game curve
- Waves 1–3 spawn ~35% fewer enemies; waves 4–6 slightly reduced
- Enemy spawn intervals increased significantly in early waves — roughly double the gap in waves 1–4

### UI
- Tower skill button is now hidden until **Combat Training** is researched
- Monkey hut tooltip overhauled: each monkey gets its own bordered card with every control on a separate row — no more cramped wrapping
- Round-robin targets each appear on their own line
- Tooltip is wider and scrollable to fit all monkey controls
